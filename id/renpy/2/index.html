# 打包安卓apk

相对麻烦,在Gura发布正式版时会构建安卓版

下载java1.8open in new window

安装java1.8全部按默认配置来

win+r输入cmd并打开,在cmd输入cd C:\Program Files\Java\jdk1.8.0_331\bin

最小化到任务栏

右键此电脑选择属性,打开高级系统设置,点击环境变量

在系统变量一栏点击新建并输入:

变量名:JAVA_HOME
变量值:C:\Program Files\Java\jdk1.8.0_331\bin

点击确定

然后在系统变量里选择path点编辑

在path点新建输入%JAVA_HOME%\bin并回车

注意

从现在开始请使用VPN

不然下载包时会失败

打开刚刚的cmd,在其中输入java -version,如果提示:

java version "1.8.0_331"
Java(TM) SE Runtime Environment (build 1.8.0_331-b09)
Java HotSpot(TM) 64-Bit Server VM (build 25.331-b09, mixed mode)

就成功了

打开renpy点击右侧的安卓

点击左侧的安装SDK并创建密钥,全部选是

在左侧选择配置,给你的应用配置名字

配置好后点击Universal APK选择构建应用包,等待即可

提醒

建议在白天构建,晚上的网络不稳定
